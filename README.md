There are 2 kind of search technique available
1) keyword search
2) semantic search

   1) keyword search uses direct test search like traditional database and etc.
   2) semantic search did the vector search and find the nearest match
      It uses 2 technique
      a) ucledian distance
      b) cosine similarity
